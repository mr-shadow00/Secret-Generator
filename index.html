env var generator

BETTER_AUTH_SECRET

A high-entropy key Better Auth uses for encryption, signing, and hashing. Generated locally with the Web Crypto API — nothing here ever leaves your browser.

.env 256-bit
BETTER_AUTH_SECRET=generating…

Use a different secret per environment (dev, staging, production). Rotating it will invalidate existing sessions and encrypted data — use BETTER_AUTH_SECRETS (plural) if you need a non-destructive rollover.

or from your terminal
$openssl rand -base64 32
$npx @better-auth/cli secret
